ServletJSPMVC
=============

Simple sample of Java Servlet and JSP web apps using MVC pattern

This simple project is about:
- Listener
- JSP
- Servlet
- MVC
- Scope _(Webaware)_

IDE: Eclipse JEE

Server: apache-tomcat-5.5.17

Dependencies:
- mysql-connector-java-5.1.34
- Apache commons-logging-1.2-bin
